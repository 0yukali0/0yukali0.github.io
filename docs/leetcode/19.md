title: 019 Remove nth node from the end of list
---
雙指針，分為快慢。
確保快針為空時，慢針在N-1。
```
delete 2
element: [fake]-[0]-[1]-[2]
origin:  slow/fast
expect:         slow        fast
move:    slow           fast
```

```go
func removeNthFromEnd(head *ListNode, n int) *ListNode {
    dummyNode := &ListNode{
        Val: -1,
        Next: head,
    }
    // origin stage
    slow, fast := dummyNode, dummyNode
    // move stage
    for i := 0; i <= n; i++ {
        fast = fast.Next
    }
    // expect stage
    for fast != nil {
        fast = fast.Next
        slow = slow.Next
    }
    tmp := slow.Next
    slow.Next = tmp.Next
    tmp.Next = nil
	return dummyNode.Next
}
```
